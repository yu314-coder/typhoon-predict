No single best model — v23 takes four of six, v26 takes two

Every version scored the same way: 10-seed ensembles on the 3,763 WP+EP test windows of 2020–2025, each intensity channel masked with its own validity flag. v23 (temporal steering stack) is the best track model and the only gain in this project that survived a paired-storm bootstrap. v26 (ocean-heat patch → CNN → intensity head) wins maximum wind and RMW. Reporting either one as "the best model" on its own would be a claim the numbers do not support.

Separating the retrain from the ocean

v26 is a fresh training run, and a fresh run moves the numbers on its own. Crediting the ocean with the whole v25→v26 gap would have been wrong — on pressure it would have been entirely wrong.

Maximum wind is the only column where the ocean term carries the gain. Pressure improved by 0.18 hPa from v25 — and every bit of that was the retrain; the ocean patch contributed +0.02. RMW and the wind radii move by less than the run-to-run spread and cancel in sign, so they are null.

The map shows v23, not v26

The world map that goes with this page draws v23, because the map draws tracks and v23 is the track winner at 435.0 km. Drawing v26 there would add nothing: its ocean token reaches only the intensity decoder, and an init assertion proved the track output is bit-identical with the token on or off, so a v26 track map would be the v25 map redrawn (452 km vs 452 km, a difference with no ocean content in it).

Separately, and this one was a mistake: the Colab runtime idled out before v26's 20 checkpoints were pulled, and /content is wiped on disconnect, so they are gone. Its numbers survive only because the results JSON was saved first. The training script now mirrors every improved checkpoint to Drive.

Bars are scaled within each panel's own range so differences of a few tenths stay visible; read the printed number, not the bar length, for magnitude. Lower is better throughout.

A 0.86 kt gain on a 17 kt base is about 5%. It has not yet been through a paired-storm bootstrap, so it is a measured difference between two matched 10-seed ensembles, not yet a significance-tested result. This project has already seen one apparent 8 km effect dissolve into seed noise under that test.
