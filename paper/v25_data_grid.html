The test-set gap is largely closed. SHIPS alone covered 7% of WP+EP test windows because it stops in 2021. Adding AOML ocean heat (2013–2026) brings the merged set to 82% — 8,434 of 10,227. The two sources are near-complementary there: 108 windows have SHIPS only, 6,789 have ocean only.
The joins corroborate each other. SHIPS carries COHC and AOML carries Ocean_Heat_Content — the same quantity from independent pipelines, matched by different keys. On the 8,053 windows where both exist they correlate at r = 0.798 with means 71.4 vs 73.8 kJ/cm². Nothing else in this dataset had been validated against an external reference.
Coverage is patchy, not uniform. 47,556 WP+EP windows still have no environmental features — almost all pre-2013, outside both sources. v25 must treat "environmental features present" as a first-class input condition using the mask, not assume they exist. Training only on covered windows would discard half the WP+EP data, and the learning curve says storms are the scarce resource.

Ocean heat shows 46% in 2013 because AOML's advertised start (2012-08-27) is wrong — 2012 is entirely NaN over our box and early 2013 is incomplete. Probed year by year rather than trusted.

Basin falls to 40% in 2026: NCEP R1 stops at 17 March. SHIPS never exceeds ~60% even in its best years because it is western Pacific only while these rows count WP+EP.

ERA5 remains absent by choice — ~11–16 h of transfer, deliberately deferred.
