TrackFormer · where the error actually is

Four curves that decided what to build next

Every number here is measured on the same held-out set — WP+EP storms from 2020 on, 3,763 full-horizon windows, five-seed ensembles. The models never saw these years in training.

Reading these together. Chart 1 says better flow prediction is worth ~115 km overall; charts 2 and 4 say more data and more architecture are worth ~18 km and ~0 km respectively. Chart 3 says the remaining error is a genuine hedge under uncertainty, not a correctable bias.

Oracle values are a first-order estimate: the weights were trained against the model's own flow-error distribution, not a perfect one, so the true ceiling could sit either side of the number shown. NHC official figures are the published five-year mean and are not a homogeneous comparison — different sample, different basins — so read them as scale, not as a head-to-head.
